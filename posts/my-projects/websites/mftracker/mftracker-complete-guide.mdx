## Limitations & Disclaimers

### What MFTracker Is NOT
- **Not a Performance Tracker** - Doesn't show historical returns
- **Not Financial Advice** - Provides analysis, not recommendations
- **Not a Trading Platform** - Can't execute transactions
- **Not a Replacement for Advisors** - Complement, not substitute

### Important Considerations
- Analysis is **hypothetical** based on assumptions
- Uses broad market/sector data, not detailed fund holdings
- Requires accurate input data for reliable results
- Browser storage can be cleared (backup exports)
- Should be used alongside professional advice

### Data Accuracy
- NAV data sourced from public APIs
- Fund details may have slight delays
- User-entered data accuracy is critical
- Regular updates recommended
